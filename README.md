## 🏢 Bl@wong Company Profile

Website company profile **Bl@wong** berbasis Laravel yang dirancang untuk menampilkan informasi bisnis secara profesional dan menarik.

🔗 Live Demo: https://blawongcompany.netlify.app/

### ✨ Fitur Utama
- 🏬 Halaman profil perusahaan
- 🧳 Showcase produk kulit
- 🛠️ Informasi layanan
- 📞 Halaman kontak bisnis
- 📱 Responsive design (mobile-friendly)

### 🛠️ Teknologi yang Digunakan
- Laravel (Backend)
- Blade Template
- HTML, CSS, JavaScript
- Bootstrap (UI Framework)
- Netlify (Deployment)

### 🎯 Tujuan Project
Project ini dibuat untuk:
- Menampilkan identitas brand Bl@wong secara profesional
- Mempermudah promosi produk dan layanan
- Menjadi media komunikasi dengan calon klien

### 🚀 Deployment
